Задание:

Создать микросервис на NestJs в docker-compose.
docker-compose должен содержать в себе:
postgresql
redis

Библиотеки, которые необходимо добавить:
typeorm 
ioredis

Функциональные требования:
Бд должна содержать в себе
пользователя

имя (до 300 символов)
фамилия (до 300 символов)
email
телефон

адрес
город (до 300 символов)
улица (до 300 символов)
номер дома (число)
тип адреса: домашний / рабочий

Один пользователь может иметь 2 адреса: Домашний и Рабочий

Пользователя можно создать
Пользователя можно редактировать
Пользователя можно удалить

Адрес можно создать
Адрес можно редактировать
Адрес можно удалить

Все контроллеры используют swagger
Предусмотреть большой объем бд (~ 1 000 000 пользователей)


Поиск пользователей по ID адреса (возвращает все данные пользователей прикрепленных к этому адресу) прикрепить кэш на REDIS

Можно получить список пользователей с поиском по имени+фамилии или по email (возвращает все данные пользователей + адреса) прикрепить кэш на REDIS
